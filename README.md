## 📌 Overview

Shivangi's AI Movie Recommender is a content-based recommendation system that suggests movies similar to a user's selection using machine learning techniques.

The recommendation engine analyzes movie metadata such as genres, keywords, cast, crew, and overview text to identify similarity patterns and generate accurate recommendations in real time.

The application is deployed using Streamlit Cloud and designed with a clean, responsive interface for seamless user interaction.

---

## ✨ Features

- 🎯 Intelligent movie recommendations using cosine similarity
- 🧠 NLP-based feature extraction and vectorization
- ⚡ Fast recommendation generation

## 🧠 Recommendation Engine Workflow

```text
Movie Metadata
       ↓
Data Cleaning & Preprocessing
       ↓
Feature Combination
       ↓
Text Vectorization (CountVectorizer)
       ↓
Cosine Similarity Matrix
       ↓
Top Similar Movies Recommended
```

```bash

```


```bash

```
